\subsection{Detailed Description}
\subsubsection*{template$<$typename K, typename V$>$class Alveole$<$ K, V $>$}

Class to define \hyperlink{class_hashtable}{Hashtable} alveoles. 

\hyperlink{class_alveole}{Alveole} class embodies a \hyperlink{class_hashtable}{Hashtable}'s alveole. An alveole store a pair $<$k,v$>$. Alveoles are simply-\/linked elements.
